Interactive recursive split pattern canvas block with bezier curves and mouse interaction.

== Description ==

Adds a Gutenberg block that renders an interactive generative art canvas with recursive split patterns.
Fully customizable via block settings: colors, layout, curves, splitting behavior, interaction, and aspect ratio.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Run `npm install && npm run build` inside the plugin folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the "NFT Split Pattern" block in the editor
